# Kbd

Keyboard shortcut display with a raised key appearance.

    <kbd class="kbd">⌘</kbd>
    <kbd class="kbd">K</kbd>
  </div>
</div>

```html
<kbd class="kbd">⌘</kbd>
<kbd class="kbd">K</kbd>
```

## Tokens used

- `--hem-bg-elevated` — key background
- `--hem-border` — key border
- `--hem-shadow-sm` — subtle shadow for raised look
- `--hem-font-mono` — monospace font
